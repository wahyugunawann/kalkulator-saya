## Simple Webservice Application using NuSOAP

Aplikasi sederhana menggunakan NuSOAP, terdiri file api/webservice di sisi server, dan file disisi client.

Akses Client di: `http://localhost/kalkulator-saya/web/client/contoh-1.php`
Akses Server WSDL di: `http://localhost/kalkulator-saya/web/server/api.php?wsdl`
